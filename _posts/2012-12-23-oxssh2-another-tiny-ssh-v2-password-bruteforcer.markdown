### Description

This a small tool that allow you to make a SSH2 Bruteforce Attack.

{% highlight bash %}
usage: ./oxssh2 <hostname> <port> <words.lst>
{% endhighlight %}

### Dependency

This require a `libssh2-dev` (SSH2 client-side library - development headers)
